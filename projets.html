📊 Supervision d'Infrastructure

Linux Debian Prometheus Grafana

L'objectif de ce projet était de mettre en place une solution de monitoring proactive pour surveiller l'état de santé du réseau et des serveurs afin d'anticiper les pannes et optimiser les performances.

  • Installation et configuration d'un serveur sous Linux.
  • Déploiement de Prometheus pour la collecte des métriques (CPU, RAM, Disques, Trafic réseau).
  • Mise en place d'agents (Node Exporter) sur les machines cibles.
  • Création de tableaux de bord personnalisés avec Grafana.
  • Configuration d'alertes en cas de dépassement de seuils critiques.

🎫 Gestion de Parc & Ticketing (GLPI)

GLPI Windows Server 2019 Active Directory (LDAP)

Déploiement d'une solution centralisée de gestion de parc informatique (ITSM) couplée à un environnement Microsoft. Ce projet vise à améliorer le support utilisateur, centraliser les identités et cartographier l'infrastructure.

  • Création et administration d'un domaine Active Directory sur Windows Server 2019.
  • Installation d'un environnement web pour héberger la solution GLPI.
  • Interconnexion LDAP entre l'AD et GLPI pour l'authentification centralisée des utilisateurs.
  • Mise en place d'un système d'inventaire automatisé pour remonter les équipements du domaine.
  • Configuration des règles de Helpdesk et gestion des profils techniques.
